Initial Developer Setup Guide

1
Get YouTube API Keys (Google Cloud)

Go to Google Cloud Console, create a project, enable the YouTube Data API v3, and create an OAuth 2.0 Client ID (select "Web Application"). Add the redirect URL shown on the left under "Authorized redirect URIs".

2
Get Meta App Keys (Facebook / Instagram)

Go to Meta for Developers, create an app, add the **Facebook Login** product, and select "Web" platform. Set the redirect URL shown on the left. Retrieve your App ID and App Secret from the Dashboard Settings.

3
Turn off Demo Mode & Post!

Save the credentials inside Settings, connect your accounts inside the Accounts tab, select your channels on the Publish desk, upload your video, and publish live instantly!
